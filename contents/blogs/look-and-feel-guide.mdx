## Why a Consistent Look & Feel Matters

In Pressbooks project, consistent styling across all chapters and components enhances visual appeal and improves user experience and accessibility. This guide serves as a comprehensive reference for styled components used in our Pressbooks projects. It helps student developers in the OER Lab to maintain visual harmony across the book by outlining key design principles and reusable styled components.

## Our Approach

To ensure a consistent and accessible design across Pressbooks project, this guide is presented in two complmentary parts:

1. **This Blog Post** - This post outlines the core principles behind our visual design and sets the standards for typography, color usage, and component styling. You'll find detailed guidance on each CSS styled component in the **Pressbooks Template**.
2. **Pressbooks Template** - A template displays all styled components in a live Pressbooks environment. It provides a clear, practical view of how each element should look and function. It also serves as the source of the custom CSS, making it easy to copy and implement styles into your own projects.

**\*Note:** Please use both this **Blog Post** and **Pressbooks Template** together for proper implementation.

### Learning Objectives

The Learning Objectives textbox is a styled component designed to highlight the expected learning outcomes for readers. It is typically placed at the beginning of a part.

**\*Note:** In Pressbooks, content is organized hierarchically into Parts and Chapters. Each Part acts as a broader section that contains multiple related Chapters. For example, you might have:

- Part 1
  - Chapter 1.1
  - Chapter 1.2
  - Chapter 1.3

}

  display: flex;
  align-items: center;
  width: fit-content;
  padding: 1rem;
  border-radius: 0.375rem;
  box-shadow: 0 4px 6px -1px rgba(0, 0, 0, 0.1),
    0 2px 4px -2px rgba(0, 0, 0, 0.1);

      cursor: pointer;
      filter: blur(8px);
      transition: all cubic-bezier(0.4, 0, 0.2, 1) 300ms;

### Table

The Table component is styled to create a bordered and horizontally centered table layout. Text contents in the table headers (`<th>`) are center-aligned and when the content inside table cells begins to shrink too much based on screen size, a horizontal scroll bar automatically appears.

![table component](/look-and-feel-guide/table.png)

```
.table-wrapper {
  overflow-x: auto;
  width: 100%;
  table {
    width: 100%;
    table-layout: auto;
    th {
      text-align: center !important;
    }
  }
}
```

**\*Note:** The **Malala Theme** in Pressbooks provides several built-in CSS classes that influence the table appearance by default:

- `table.grid`: Adds a black border around all table cells.
  ```
  table.grid {
    border: 1px solid; border-color: black;
  }
  ```
- `table.aligncenter`: Centers the table horizontally within the page.
  ```
  table.aligncenter {
    margin-left: auto;
    margin-right: auto;
  }
  ```
- For more information on how spacing and structural alignment are handled, please refer to the [Layout & Spacing](#layout--spacing) section
- Please use these default CSS classes when creating a table, using `<table class="grid aligncenter">`

## How to Create Images and Diagrams

To maintain consistency and quality in visual elements across Pressbooks projects, we recommend using Canva for general image creation and GeoGebra for diagrams and graphs.

### Image Creation in Canva

1. Visit free image sites like [Pixabay](https://pixabay.com/), [Freepik](https://www.freepik.com/), or similar platforms. Search for suitable images and download your selection.
2. Go to [Canva](https://www.canva.com/) and log in using the Centennial College OER Lab account:
     ![canva login button](/look-and-feel-guide/canva-login-1.png)
     ![canva login with OERLab email account](/look-and-feel-guide/canva-login-2.png)
     ![canva login with OERLab email password](/look-and-feel-guide/canva-login-3.png)
3. Click “Create a design” and choose "Custom Size" and enter 700 x 500 px (recommended size)
   ![canva create design](/look-and-feel-guide/canva-create-1.png)
   ![canva create custom size](/look-and-feel-guide/canva-create-2.png)
4. Click on the “Uploads” tab and upload your previously downloaded image
   ![canva upload the image](/look-and-feel-guide/canva-create-3.png)
5. Adjust, crop, and edit as needed to fit your design goals
6. Click the “Share” button, then choose “Download”
   ![canva download the image](/look-and-feel-guide/canva-create-4.png)
7. Navigate to the page in your Pressbooks project where you want to insert the image
8. Click where you’d like to place the image and choose “Add Media”
   ![canva add image to the Pressbooks](/look-and-feel-guide/canva-insert-1.png)
9. Go to “Upload Files”, click “Select Files”, and upload your image
   ![canva upload image to the media library](/look-and-feel-guide/canva-insert-2.png)
10. Once uploaded, click “Insert” at the bottom-right of the screen
    ![canva insert image](/look-and-feel-guide/canva-insert-3.png)

### Diagram Creation in GeoGebra

1. Visit [GeoGebra](https://www.geogebra.org/) and click "Start Calculator"
   ![geogebra click start calculator](/look-and-feel-guide/geogebra-start.png)
2. Setting up the workspace
   - Go to setting -> Basic -> Set the x-axis and y-axis ratio to 1:1
     ![geogebra setting x-axis and y-axis ratio to 1:1](/look-and-feel-guide/geogebra-setting-1.png)
   - Go to setting -> xAxis setting -> Set the distance to 1
     ![geogebra setting x-axis distance to 1](/look-and-feel-guide/geogebra-setting-2.png)
   - Go to setting -> yAxis setting -> Set the distance to 1
     ![geogebra setting y-axis distance to 1](/look-and-feel-guide/geogebra-setting-3.png)
3. Select "Tools" tab on the left side
4. Select "Vector" tool under "Lines" to draw primary vector
   ![geogebra draw vector lines](/look-and-feel-guide/geogebra-vector.png)
5. Select "Circular Sector" tool under "Circles" to draw an angle
   ![geogebra draw angle](/look-and-feel-guide/geogebra-circular.png)
6. Select "Segment" under "Lines" to draw a reference line
   ![geogebra draw segment lines](/look-and-feel-guide/geogebra-segment.png)
7. Select "Text" under "Media" to insert labels. For vector labels, use LaTeX syntax for an overhead arrow (`\overrightarrow{A} for vector A`). Replace A with your vector's label
   ![geogebra insert text label](/look-and-feel-guide/geogebra-text.png)
8. Select the line and set the "Line Thickness" to 8
   ![geogebra set the line thickness to 8](/look-and-feel-guide/geogebra-line-thickness.png)
9. For angles and its labels and lines, set the color to gray. The theta symbol (θ) should be black and bold
   ![geogebra set the angle to gray and its label color to black bold](/look-and-feel-guide/geogebra-angle-color.png)
10. Set the reference line color to red and make it dotted line by selecting the line, and choose "Line Style" to dotted line
    ![geogebra set the reference line color to red and dotted line](/look-and-feel-guide/geogebra-dotted-line.png)
11. Set the position vectors color to blue
    ![geogebra set the vector line color to blue](/look-and-feel-guide/geogebra-vector-color.png)
12. To hide auto-generated labels (like A, B, C, D), click the label and unselect "Show Object"
    ![geogebra hide labels](/look-and-feel-guide/geogebra-hide-label.png)
13. Use Serif font for non-LaTeX texts and set text size to medium, and make it bold
14. Export and cropping
    - Click "Hamburger" menu at the top left side.
    - Click "Export Image" and click "Download" button
      ![geogebra click export image button](/look-and-feel-guide/geogebra-export-1.png)
      ![geogebra click download image button](/look-and-feel-guide/geogebra-export-2.png)
    - After download the image, open the image and crop the image to one small square away from the last element for a clean and consistent layout

## How to Use Look and Feel Guide and Pressbooks Project Template

This **Blog Post**, along with the accompanying **Pressbooks Template**, serves as both a visual reference and a styling resource for creating consistent, accessible, and professional-quality content.

Together, they provide everything you need to build with confidence:

- Use this **Blog Post** to understand the design principles and access the necessary CSS code
- Use the **Pressbooks Template** to copy styles and components directly into your own project

**\*Note:** Please do not clone the **Pressbooks Template** directly. Cloning can cause issues like the copyright statement that cannot be easily changed later.

\*A link to the **Pressbooks Template**: [Pressbooks Template](https://ecampusontario.pressbooks.pub/pressbookstemplate/)

### Steps to apply custom CSS styles in Pressbooks Template

1. Create a new Pressbooks project
2. Open the **Pressbooks Template** by using the above link
3. Copy the custom CSS from the **Pressbooks Template**
   - Navigate to the Custom CSS section within the **Pressbooks Template** dashboard (`Apperance` -> `Custom Styles`)
   - Select and copy all the custom CSS styles within the **Your Web Styles** area
4. Paste it into new Pressbook project
   - In your new Pressbooks project dashboard, go to the `Apperance` -> `Custom Styles`
   - Paste the copied CSS styles into the **Your Web Styles** area
   - Click **"Save** button
     ![apply custom CSS in the Pressbooks Template and Save it](/look-and-feel-guide/apply-custom-css.png)
5. Always remember to check actual page to confirm whether the custom CSS styles have been applied correctly.

### Import the Pressbooks Template to a new Pressbooks project

You can selectively import specific chapters or pages from the **Pressbooks Template**.

1. Open your new Pressbooks project dashboard
2. Click "Import" tab on the left side
3. Select "Web page or Pressbooks webbook (.html or URL)
4. Select "Import from URL"
5. Enter Source URL of the **Pressbooks Template**
6. Click "Begin Import" button
   ![import pressbook template](/look-and-feel-guide/import-template.png)

**\*Note:** This method is not recommended and it is optional. Primary method for styling is copy and paste the custom CSS styles in the **Pressbooks Template**.
